# FKPrizeWheel
肥客大转盘抽奖fk-prize-wheel浏览器原生组件Web Components API
+ 组件特性：原生组件Web Components API

## 组件安装：
1. html引用（推荐）
```html
<script type="module">
  import './fk-prize-wheel.js';
</script>
``` 

2. 使用文件引入
```html 
<script type="module" src="./fk-prize-wheel.js"></script>
```

3. vue或react项目引用
```js 
import './fk-prize-wheel.js';
```
